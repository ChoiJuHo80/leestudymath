# 이공 수학 공부방 블로그 & LMS 플랫폼

이 프로젝트는 초·중등 전문 **이공 수학 공부방**의 프리미엄 블로그 스타일 홈페이지와 학부모/학생 포털, 그리고 원장(선생님)을 위한 학생 관리(LMS) 및 실시간 1:1 상담 메신저를 결합한 통합 웹 플랫폼입니다.

## 📂 파일 구조

- **`index.html`**: 메인 페이지 마크업 및 모달 레이아웃 (공지사항, 커리큘럼, 수업 스케줄, 자료공유, 관리자 모드, 마이클래스 등)
- **`styles.css`**: 현대적이고 고급스러운 디자인 시스템이 반영된 바닐라 CSS 스타일시트 (Glassmorphism, 애니메이션 등)
- **`main.js`**: 공지사항 CRUD, 학생 정보 관리, 출결 안심 SMS 알림 시뮬레이션, 실시간 1:1 채팅 시뮬레이션 등 전체 프론트엔드 비즈니스 로직
- **`package.json`**: 로컬 서버 구동을 위한 Vite 환경 설정 및 패키지 관리 파일

---

## 🚀 로컬 서버 실행 방법

현재 로컬 개발 서버가 이미 작동 중입니다! 아래 주소를 통해 접속하실 수 있습니다.

* **접속 주소**: [http://localhost:5173/](http://localhost:5173/)

만약 나중에 서버를 다시 구동하려면 터미널에 다음 명령어를 실행하십시오:

```bash
# 의존성 패키지 설치 (최초 1회)
npm install

# 로컬 개발 서버 시작
npm run dev
```

## ✨ 핵심 기능 및 테스트 안내

### 1. 📢 공지사항 (CRUD)
- 최상단 고정(Pin to Top) 설정이 가능한 반응형 공지사항 보드입니다.
- 우측 상단 **[관리자 로그인]** 버튼 클릭 후 로그인(비밀번호: `9999`) 시 새 공지사항 등록, 기존 공지사항 수정/삭제가 가능합니다.

### 2. 🎒 학생 관리 (LMS Dashboard) - 관리자 모드 전용
- **원생 등록 및 관리**: 새로운 원생의 나이, 학교, 학부모 연락처, 수업 요일별 시간표, 진도 현황 및 피드백을 기록합니다.
- **실시간 등하원 안심 문자**: 등원/하원 버튼 클릭 시 학부모 안심문자 발송 알림이 즉시 전송 시뮬레이션됩니다.
- **과제 출제**: 각 학생별로 마크다운 형식의 일일 숙제를 출제할 수 있습니다.
- **1:1 피드백 메신저**: 학생/학부모 포털로부터 오는 메시지를 실시간으로 확인하고 즉시 답변할 수 있는 전용 모달 창을 제공합니다.

### 3. 🔑 마이클래스 (Student & Parent Portal) - 학생/학부모 로그인 전용
- 우측 상단 **[학생/학부모 로그인]** 버튼을 통해 로그인합니다.
  - **테스트용 계정**: 이름 `김민준` / 연락처 `010-9876-5432`
- **학습 대시보드**: 아이의 현재 진도 상황, 시간표, 원장 선생님의 개별 지도 피드백을 확인합니다.
- **과제 체크리스트**: 원장 선생님이 내주신 숙제를 확인하고, 완료한 경우 체크하면 선생님 대시보드에 완료 건수가 실시간 업데이트됩니다.
- **1:1 실시간 메신저**: 원장 선생님과 실시간으로 대화를 나눌 수 있는 톡 형식의 메신저입니다.

### 4. 📂 자료공유 센터
- 공부방에서 무료로 배포하는 핵심 공식집 및 모의고사 문제를 다운로드할 수 있으며, 실제 다운로드 대기 애니메이션과 토스트 메시지가 나타납니다.
